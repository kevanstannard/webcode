JavaScript Assignments

Although each of these assignments will need varying degrees of HTML, CSS and DOM manipulation skills, they are designed to focus on JavaScript without too much emphasis on the everything else.

Write the solution using whatever JavaScript technology stack you like but focus on implementing them with a clean architecture.

  • Audio Player

    Create a mock audio player to play, pause and cycle through a song list.

  • Cumulative Progress Bars

    Display multiple independent progress bars plus a cumulative total progress bar.

  • Filtered Grid

    A simple grid that may be selectively filtered.

  • Movie Publisher

    A list of movies that need to be processed sequentially (asynchronously).

  • Multiple Choice Quiz

    A multiple choice quiz with a different page for each question.

  • Sign Up Process

    A sign-up process with multiple steps.

  • Tag List

    Enter tags names in a text box and display as a tag list.

  • TODO List

    Enter TODO items in a text box and display in a TODO List.
