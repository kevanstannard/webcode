--- layout: assignment title: Tag List ---

Your task is so write a simple Tag List.

A new tag is added by entering it's name in the text box and pressing Enter.

Tags are listed one after another in the order they were added.

Each tag has an X that removes the tag.

Notes

The tag list should work on both small and large displays and should adjust it's width according to the size of it's container.

Enhancements

Feel free to enhance your application with nicer styles or improved functionality.
