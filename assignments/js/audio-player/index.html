--- layout: assignment title: Audio Player ---

Your task is so write a mock audio player that simulates playing a list of songs.

A list of audio tracks should be provided as a JavaScript array. The duration is provided in seconds, but you can use short durations for this assignment.

{% highlight javascript %} var tracks = [ { name: "Imagine", artist: "John Lennon", duration: 15 }, { name: "We are the Champions", artist: "Queen", duration: 20 }, { name: "Hey Jude", artist: "The Beatles", duration: 10 } ]; {% endhighlight %}

When the application starts it should immediately start playing the first track.

When playing it should show Pause, Previous and Next buttons.

When paused it should show Play, Previous and Next buttons and the song title in the display should indicate that it's paused.

Notes

Use a JavaScript timer to simulate the playing of the song and update the progress bar.

Display the current elapsed time of the song in minutes and seconds alongside the progress bar.

When the last song finishes, the application should start playing the first song again.

The application should work on both small and large displays and should adjust it's width according to the size of it's container.

Enhancements

Feel free to enhance your application with nicer styles or improved functionality.
